Elemento Descripción Ejemplo Información
<html> El elemento < HTML > (o elemento HTML raiz) representa la raiz de un documento HTML. El resto de elementos descienden de este elemento. Documentacion
<head> El <header> es un elemento que representa contenido introductorio, normalemnte un grupo de ayudas introudctorias o de navegacion puede contener algunos elementos de encabezado pero también un logotipo, un formulario de búsqueda, el nombre del autor y otros elementos. Documentacion
<meta> El < meta> es un elemento que representa metadatos que no pueden ser representados por otros metaelementos HTML
<body> Es el cuerpo del documento Documentacion
<div> Elemento generico Documentacion
<input> Crea controles interactivos para formularios basados en la web Documentacion
<span> Contenedor genérico en línea para redactar contenido, que no representa inherentemente nada Documentacion
<nav> Representa una sección de una página cuyo propósito es proporcionar enlaces de navegación Documentacion
<aside> Se utiliza para definir contenido secundario o complementario dentro de un elemento principal, como un artículo o una sección de una página web, generalmente se considera como información que es relevante para el contenido principal pero no es esencial para su comprensión. Puede contener elementos como anuncios, widgets, notas de la autora, enlaces relacionados o cualquier otro contenido adicional. Documentacion
<footer> representa un pie de página Documentacion
<main> Representa el contenido dominante del cuerpo de un documento Documentacion
<article> composición independiente en un documento Documentacion
<section> Se utiliza para dividir una página web en secciones temáticas o agrupar contenido relacionado. Documentacion
<h1-h6> Representan jerarquia en titulos siendo h1 la mas alta y h6 la mas baja Documentacion
<p> Distribuye el texto en párrafos Documentacion
<br> Produce un salto de línea en el texto Documentacion
<hr> Representa una ruptura temática entre elementos a nivel de párrafo Documentacion
<wbr> indicar un posible lugar donde el navegador web puede realizar un salto de línea o una división automática de palabras cuando sea necesario para ajustar el texto en la ventana del navegador. La abreviatura "wbr" significa "word break opportunity" o "oportunidad de ruptura de palabra". Documentacion
<sup> se utiliza para crear texto o elementos que deben mostrarse como superíndices en una página web. Los superíndices son caracteres o números que aparecen ligeramente elevados por encima de la línea de texto normal y se utilizan comúnmente para representar exponentes, índices, notas a pie de página, marcas de registro y otras notaciones similares. Documentacion
<button> se utiliza para crear un botón interactivo en una página web que los usuarios pueden hacer clic para realizar una acción o activar una función JavaScript. Los botones son elementos esenciales para la interacción del usuario en aplicaciones web y formularios. Documentacion
<a> crea un hipervínculo a páginas web Documentacion
<title> define el título del documento Documentacion
<sub> Especifica el texto en línea que debe mostrarse como subíndice únicamente por motivos tipográficos Documentacion
<sup> Especifica el texto en línea que se mostrará como superíndice únicamente por motivos tipográficos Documentacion
<strong> Indica que su contenido tiene una gran importancia, seriedad o urgencia. Documentacion
<em> Marca el texto que tiene énfasis. Documentacion
<mark> Representa texto que está marcado o resaltado para fines de referencia. Documentacion
<q> Indica que el texto adjunto es una cita breve en línea. Documentacion
<blockquote> Indica que el texto adjunto es una cita extendida. Documentacion
<cite> Se usa para describir una referencia a un trabajo. Documentacion
<dfn> Indica el término que se define dentro del contexto de una frase u oración de definición. Documentacion
<abbr> Representa una abreviatura o un acrónimo. Documentacion
<small> Representa comentarios secundarios y letra pequeña, como derechos de autor y texto legal, independiente de su estilo. Documentacion
<ins> Representa un rango de texto que se ha agregado a un documento. Documentacion
<del> Representa un rango de texto que se ha eliminado de un documento. Documentacion
<s> Muestra el texto tachado o con una línea que lo atraviesa. Documentacion
<code> Muestra su contenido con estilo para indicar que el texto es un fragmento corto de código de computadora Documentacion
<samp> Se utiliza para encerrar texto en línea que representa la salida de muestra de un programa de computadora Documentacion
<kbd> Representa un tramo de texto en línea que indica la entrada textual del usuario desde un teclado, voz o cualquier otro dispositivo de entrada de texto Documentacion
<pre> Representa texto preformateado que debe presentarse exactamente como está escrito en el archivo HTML Documentacion
<address> Proporciona información de contacto para una persona o para una organización Documentacion
<img> Incrusta una imagen en el documento. Documentacion
<table> Representa datos tabulares, es decir, información presentada en una tabla bidimensional compuesta por filas y columnas de celdas que contienen datos. Documentacion
<progress> El elemento <progress> se utiliza para representar el progreso de una tarea en un formulario o en cualquier otro contexto donde el progreso se pueda medir. Puede tener un valor mínimo y máximo, y se muestra visualmente como una barra de progreso que se llena a medida que avanza la tarea. Documentacion
<iframe> El elemento <iframe> se utiliza para incrustar contenido externo, como páginas web, documentos PDF o videos, dentro de una página web. Permite crear una ventana incrustada dentro de la página principal y mostrar contenido de otro origen. Documentacion
<audio> El elemento <audio> se utiliza para incrustar archivos de audio en una página web. Puede reproducir archivos de audio como MP3, Ogg, WAV y más, proporcionando controles de reproducción para los usuarios. Documentacion
<video> El elemento <video> se utiliza para incrustar archivos de video en una página web. Puede reproducir archivos de video en varios formatos, como MP4, WebM, y otros, y proporcionar controles de reproducción para los usuarios. Documentacion
<canvas> El elemento <canvas> se utiliza para dibujar gráficos, imágenes y animaciones en una página web utilizando JavaScript. Proporciona un lienzo en blanco donde puedes realizar dibujos y manipulaciones gráficas. Documentacion
<select> El elemento <select> se utiliza para crear listas desplegables en formularios web. Permite a los usuarios seleccionar una opción de una lista de opciones predefinidas. Documentacion
<textarea> El elemento <textarea> se utiliza para crear áreas de texto de múltiples líneas en formularios web. Los usuarios pueden escribir texto largo en estas áreas, como comentarios o mensajes. Documentacion
<time> El elemento <time> se utiliza para representar una fecha o una hora en un formato específico. Puede ser útil para marcar eventos, publicaciones de blog, horarios y más. Documentación
<data> El elemento <data> se utiliza para contener valores de datos que pueden ser utilizados por scripts. Es útil para almacenar información en el lado del cliente y acceder a ella mediante JavaScript. Documentación
<datalist> El elemento <datalist> se utiliza en conjunto con un elemento <input> para proporcionar una lista de sugerencias o opciones al usuario mientras ingresa datos en un campo de entrada. Documentación
<embed> El elemento <embed> se utiliza para incrustar contenido multimedia, como audio o video, en una página web. Permite que los navegadores reproduzcan el contenido directamente en la página. Documentación
<fieldset> El elemento <fieldset> se utiliza para agrupar elementos de formulario relacionados dentro de un formulario web. Puede contener etiquetas, inputs, botones de opción y más. Documentación
<figure> El elemento <figure> se utiliza para representar contenido independiente, como imágenes, ilustraciones, gráficos, etc., dentro de un documento HTML. Se puede utilizar junto con <figcaption> para proporcionar una descripción o leyenda. Documentación
<link> El elemento <link> se utiliza para conectar un documento HTML con una hoja de estilo externa (por ejemplo, CSS) o con otros recursos, como fuentes o iconos. También se utiliza para especificar el tipo de relación entre el documento actual y el recurso enlazado. Documentación
<base> El elemento <base> se utiliza para especificar una URL base para todas las URLs relativas en un documento HTML. Esto es útil cuando se trabaja con enlaces relativos en lugar de URLs absolutas. Documentación
<script> El elemento <script> se utiliza para incluir código JavaScript en una página web. Puede estar contenido en el documento HTML o referirse a un archivo externo. Documentación
<noscript> El elemento <noscript> se utiliza para proporcionar contenido alternativo o instrucciones para los usuarios que tienen JavaScript desactivado en su navegador. El contenido dentro de <noscript> se mostrará si el navegador no admite JavaScript o si está desactivado. Documentación
<form> El elemento <form> se utiliza para crear un formulario en una página web. Puede contener elementos de formulario como inputs, botones de envío y más. Los datos ingresados por el usuario en un formulario se envían al servidor para su procesamiento. Documentación
<hgroup> El elemento <hgroup> se utiliza para agrupar varios elementos de encabezado (<h1> a <h6>) que representan un título o subtítulo en una página web. Ayuda a mejorar la accesibilidad y la estructura del documento. Documentación
<i> El elemento <i> se utiliza para aplicar estilo o énfasis a partes del texto sin darle un significado especial. Anteriormente se usaba para texto en cursiva, pero su uso ha evolucionado para enfatizar el texto de diferentes maneras. Documentación
<map> El elemento <map> se utiliza junto con la etiqueta <area> para definir un mapa de imágenes en una página web. Permite crear áreas clicables en una imagen y vincularlas a diferentes destinos. Documentación
<math> El elemento <math> se utiliza para representar contenido matemático en una página web utilizando notación matemática, como ecuaciones y fórmulas. Se usa en conjunto con elementos como <mi>, <mo> y <msup> para estructurar el contenido matemático. Documentación
<menu> El elemento <menu> se utiliza para definir un menú de navegación en una página web. Puede contener elementos de lista (<li>) que representan elementos de menú y submenús. Este elemento se utiliza en conjunto con CSS y JavaScript para crear menús interactivos. Documentación
<meter> El elemento <meter> se utiliza para representar una medida escalar en una barra de medición dentro de un rango conocido, como la cantidad de descarga de datos, la capacidad de una batería, etc. Es útil para mostrar información de manera visual. Documentación
<object> El elemento <object> se utiliza para insertar objetos multimedia, como contenido Flash, audio o video, en una página web. Permite especificar datos, tipos MIME y más para el objeto incrustado. Documentación
<ol> El elemento <ol> se utiliza para crear listas ordenadas, donde los elementos se enumeran con números, letras u otros marcadores ordenados. Cada elemento de lista se define con la etiqueta <li>. Documentación
<output> El elemento <output> se utiliza para mostrar el resultado de un cálculo o una operación realizada por un script. Puede contener datos generados dinámicamente y se actualiza automáticamente cuando cambian los valores de entrada. Documentación
<picture> El elemento <picture> se utiliza para proporcionar múltiples fuentes de imágenes para diferentes tamaños de pantalla y resoluciones. Es útil para la implementación de imágenes responsivas. Documentación
<ruby> El elemento <ruby> se utiliza para representar texto con anotaciones de Ruby, que son pequeños caracteres o palabras que proporcionan información adicional sobre la pronunciación o significado de un carácter o palabra en un idioma diferente. Documentación
<template> El elemento <template> se utiliza para almacenar fragmentos de contenido HTML que no se deben mostrar en la página cuando se carga inicialmente. Puede clonarse y activarse mediante JavaScript para generar contenido dinámicamente. Documentación
<u> El elemento <u> se utiliza para aplicar un estilo subrayado al texto. Sin embargo, se recomienda evitar su uso para presentación y, en su lugar, utilizar CSS para estilizar el texto. Documentación
<ul> El elemento <ul> se utiliza para crear listas no ordenadas, donde los elementos se enumeran con viñetas o marcadores no ordenados. Cada elemento de lista se define con la etiqueta <li>. Documentación
<var> El elemento <var> se utiliza para representar una variable en un contexto matemático o de programación. Ayuda a identificar y dar significado a las variables dentro de un texto. Documentación
